# Smart Timetable App

A modern timetable management application built with Next.js that helps students organize classes, assignments, exams, and daily schedules efficiently.

## Features

- 🔐 Google Authentication
- 📅 Smart Timetable Management
- 📝 Assignment Tracking
- 🔔 Notifications & Reminders
- 🔥 Firebase Integration
- 🗄️ PostgreSQL Database
- 📱 Responsive Design
- 🌙 Modern UI

## Tech Stack

- Next.js
- React
- TypeScript
- Prisma ORM
- PostgreSQL
- Firebase
- NextAuth.js

## Installation

```bash
git clone https://github.com/AnshDeveloper2003/smart-timetable-app.git
cd smart-timetable-app
npm install
```

## Environment Variables

Create a `.env.local` file:

```env
NEXTAUTH_URL=
NEXTAUTH_SECRET=
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
DATABASE_URL=
NEXT_PUBLIC_FIREBASE_API_KEY=
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=
NEXT_PUBLIC_FIREBASE_PROJECT_ID=
```

## Run Locally

```bash
npm run dev
```

Open:

```
http://localhost:3000
```

## Build

```bash
npm run build
npm start
```
